# Random Password Generator

## Description

The Random Password Generator is a command-line project developed using Python that helps users generate secure and random passwords based on their preferences. The application allows users to generate multiple passwords, specify the desired password length, and choose whether to exclude letters, numbers, or special characters from the generated passwords.

The program uses Python's built-in `random` and `string` modules to create unique passwords and automatically saves them to a text file (`passwords.txt`) for easy access. This project demonstrates the use of Python fundamentals such as loops, conditional statements, string manipulation, randomization, user input handling, and file handling.

## Features

* Generate multiple passwords at once
* User-defined password length
* Option to exclude letters
* Option to exclude numbers
* Option to exclude special characters
* Option to include all character types
* Automatically saves generated passwords to a text file
* Simple menu-driven interface

## Technologies Used

* Python 3
* Random Module
* String Module

This Random Password Generator is a beginner-friendly Python project that provides a practical solution for creating customizable and secure passwords while strengthening core programming concepts and problem-solving skills.
